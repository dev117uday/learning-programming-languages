# Learning Hardhat

**Hardhat Setup**

```bash
$ npm i -g hardhat
$ npm i -D hardhat
$ npx hardhat
# to print help, again run
$ npx hardhat
# to list accounts
$ npx hardhat accounts
# to run test
$ npx hardhat test
# to compile
$ npx hardhat compile
# to run a script
$ npx hardhat run scripts/sample-script.js
# to run a local instance 
$ npx hardhat node
# If you want to connect Hardhat to this node to, for example, run a deployment script against it, you simply need to run it using --network localhost
$ npx hardhat run scripts/sample-script.js --network localhost
# you can get the sample config from https://hardhat.org/config/
```
